# 공통 컴포넌트 가이드

## 📋 개요
이 웹사이트는 **header.js**와 **footer.js** 공통 컴포넌트를 사용하여 모든 페이지에서 일관된 UI를 제공합니다.

---

## 🎯 공통 컴포넌트 구조

### 1. Header 컴포넌트 (`header.js`)

#### 기능
- 모든 페이지 상단에 고정된 네비게이션 바
- 반응형 디자인 (데스크톱/모바일)
- 스크롤 시 자동 숨김/표시 기능
- 현재 페이지 활성화 표시

#### 페이지 메뉴 구성
```javascript
Home        → index.html      (홈)
About       → about.html      (회사 소개)
AI Courses  → ai-courses.html (AI 강의 서비스)
Products    → product.html    (아이디어 상품 제조)
Contact     → contact.html    (문의하기)
```

#### 사용 방법
각 HTML 페이지에 다음 코드를 포함:

```html
<!-- Head에 스크립트 추가 -->
<script src="header.js"></script>

<!-- Body에 컨테이너 추가 -->
<div id="header-container" data-page="페이지ID"></div>
```

#### 페이지 ID 값
- `home` - 메인 페이지
- `about` - 회사 소개 페이지
- `ai-courses` - AI 강의 페이지
- `products` - 상품 제조 페이지
- `contact` - 문의 페이지

## 🔧 새 페이지 추가 방법

### 1단계: HTML 파일 생성
```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>페이지 제목 | AI Idea Manufacturing</title>
    
    <!-- Tailwind CSS -->
    <script src="https://cdn.tailwindcss.com"></script>
    <script>
        tailwind.config = {
            theme: {
                extend: {
                    fontFamily: { sans: ['Noto Sans KR', 'sans-serif'] },
                    colors: {
                        brand: { 
                            light: '#fbb656', 
                            DEFAULT: '#f9a01a', 
                            dark: '#d88810', 
                            accent: '#0f172a' 
                        }
                    }
                }
            }
        }
    </script>
    
    <!-- 공통 스타일 -->
    <link rel="stylesheet" href="styles.css">
    
    <!-- 아이콘 & 애니메이션 라이브러리 -->
    <script src="https://unpkg.com/lucide@latest"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js"></script>
    <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/ScrollTrigger.min.js"></script>
    
    <!-- 공통 컴포넌트 -->
    <script src="header.js"></script>
    <script src="footer.js"></script>
</head>
<body class="bg-slate-50 text-slate-800 font-sans overflow-x-hidden">
    
    <!-- Header Container -->
    <div id="header-container" data-page="새페이지ID"></div>

    <main id="main-content" class="pt-20">
        <!-- 여기에 페이지 콘텐츠 작성 -->
    </main>

    <!-- Footer Container -->
    <div id="footer-container"></div>

    <script type="module" src="main.js"></script>
</body>
</html>
```

### 2단계: header.js 수정
메뉴에 새 페이지 추가:

```javascript
const menuItems = [
    { id: 'home', label: 'Home', href: 'index.html' },
    { id: 'about', label: 'About', href: 'about.html' },
    { id: 'ai-courses', label: 'AI Courses', href: 'ai-courses.html' },
    { id: 'products', label: 'Products', href: 'product.html' },
    { id: 'contact', label: 'Contact', href: 'contact.html' },
    // 새 페이지 추가
    { id: '새페이지ID', label: '표시할이름', href: '새페이지.html' }
];
```

## 🎨 브랜드 컬러 시스템

프로젝트 전체에서 사용하는 통일된 컬러:

```javascript
brand: {
    light: '#fbb656',    // 밝은 주황색
    DEFAULT: '#f9a01a',  // 메인 주황색 (기본값)
    dark: '#d88810',     // 어두운 주황색
    accent: '#0f172a'    // 강조 어두운색
}
```

### 사용 예시
```html
<!-- 배경색 -->
<div class="bg-brand">주황 배경</div>
<div class="bg-brand-light">밝은 주황 배경</div>
<div class="bg-brand-dark">어두운 주황 배경</div>

<!-- 텍스트 색상 -->
<p class="text-brand">주황 텍스트</p>

<!-- 호버 효과 -->
<button class="hover:bg-brand-dark">버튼</button>
```

## ⚡ 애니메이션 라이브러리

### GSAP (GreenSock Animation Platform)
모든 페이지에서 GSAP를 사용할 수 있습니다.

#### 기본 사용법
```javascript
// 페이지 로드 시 애니메이션
window.addEventListener('load', function() {
    if (typeof gsap !== 'undefined') {
        gsap.to('.element', {
            opacity: 1,
            duration: 1,
            ease: 'power2.out'
        });
    }
});
```

#### ScrollTrigger 사용 (스크롤 애니메이션)
```javascript
gsap.to('.element', {
    opacity: 1,
    scrollTrigger: {
        trigger: '.element',
        start: 'top 80%'
    }
});
```

## 🔍 아이콘 사용 (Lucide Icons)

모든 페이지에서 Lucide 아이콘을 사용할 수 있습니다.

### 사용 방법
```html
<!-- HTML에 아이콘 태그 작성 -->
<i data-lucide="mail" class="w-6 h-6"></i>
<i data-lucide="phone" class="w-6 h-6"></i>
<i data-lucide="map-pin" class="w-6 h-6"></i>
```

### 자주 사용하는 아이콘
- `mail` - 이메일
- `phone` - 전화
- `map-pin` - 위치
- `menu` - 메뉴 (햄버거)
- `send` - 전송
- `bot` - AI/로봇
- `package-search` - 상품 검색
- `sparkles` - 반짝임

아이콘 목록: https://lucide.dev/icons

## ❓ 문제 해결

### 헤더/푸터가 표시되지 않을 때
1. `header.js`, `footer.js` 파일이 같은 폴더에 있는지 확인
2. HTML에서 스크립트 로드 순서 확인
3. 브라우저 콘솔에서 에러 메시지 확인

### 메뉴 활성화가 안 될 때
- `data-page` 속성 값이 `header.js`의 `menuItems` ID와 일치하는지 확인

### 스타일이 적용되지 않을 때
- Tailwind CDN 로드 확인
- `tailwind.config` 설정 확인
- 브랜드 컬러 설정 확인
